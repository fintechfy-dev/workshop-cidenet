# El caso lo trae tu facilitador

Este repo **no incluye ningún caso de negocio** — a propósito. Es una plantilla reutilizable: el mismo taller sirve para cualquier caso.

El caso de práctica te lo entrega tu facilitador como un documento aparte. Al iniciar el discovery:

1. **Carga ese documento en el chat** de Claude Code (arrástralo o pégalo).
2. Ejecuta `/discovery`.

El discovery te va a preguntar tu nombre y el caso, va a guardar tu copia del caso en `sessions/<tu-nombre>/caso.md`, y a partir de ahí te entrevista para construir tu especificación.

> ¿Por qué el repo no trae el caso? Para que el discovery sea un descubrimiento real: ni el repo ni Claude entran sabiendo las respuestas de antemano. Y para que el facilitador pueda cambiar el caso sin tocar el repo.
